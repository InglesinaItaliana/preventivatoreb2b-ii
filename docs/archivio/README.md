# 📦 Archivio documenti

Documenti **non più operativi** ma tenuti come **riferimento storico**: piani di
implementazione conclusi, audit-snapshot superati, analisi di sessione il cui
contenuto è stato assorbito altrove.

Non sono spazzatura — sono congelati di proposito. Si **leggono** per capire il
"perché" di una scelta passata, non si **aggiornano**.

> Regola: un doc entra qui quando il suo scopo è esaurito (piano eseguito,
> roadmap completata, snapshot datato) ma cancellarlo perderebbe contesto utile.
> Se documenta comportamento ancora **live**, in testa al file va un rimando alla
> fonte di verità attuale.
